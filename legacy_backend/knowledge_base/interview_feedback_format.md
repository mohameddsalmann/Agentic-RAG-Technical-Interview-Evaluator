# Interview Feedback Format

## Standard Evaluation Report Structure

### Header
- Candidate Name
- Role Applied For
- Seniority Level
- Problem Title
- Evaluation Date
- Evaluator: AI Agent (with human review)

### Overall Assessment
- **Overall Score**: 0-100 (weighted average)
- **Recommendation**: Strong Hire / Hire / Lean Hire / Lean No Hire / No Hire
- **Human Review Required**: Yes (always)

### Category Scores
| Category | Score | Weight |
|----------|-------|--------|
| Correctness | X/100 | 30% |
| Complexity | X/100 | 15% |
| Edge Cases | X/100 | 15% |
| Readability | X/100 | 10% |
| Maintainability | X/100 | 10% |
| Communication | X/100 | 20% |

### Complexity Analysis
- **Time Complexity**: Big-O notation with explanation
- **Space Complexity**: Big-O notation with explanation

### Strengths
- List specific technical strengths observed in the code
- Each strength should reference specific code evidence
- Maximum 5 strengths, ordered by significance

### Weaknesses
- List specific technical weaknesses observed in the code
- Each weakness should reference specific code evidence
- Maximum 5 weaknesses, ordered by severity

### Missed Edge Cases
- List specific edge cases not handled by the solution
- Reference the expected solution for comparison
- Prioritize by importance

### Retrieved Rubric Evidence
- List rubric snippets that influenced the evaluation
- Include source file name
- Include relevance score
- Show how evidence was applied

### Bias Mitigation Notes
- Confirm feedback is based on technical evidence only
- Note any bias checks performed
- Confirm no personal traits were considered

### Final Feedback
- 2-3 paragraph professional summary
- Start with overall assessment
- Discuss key strengths
- Discuss areas for improvement
- End with recommendation rationale
- Include: "Human review required before final hiring decision."

## Tone and Style Guidelines
- Use neutral, professional language
- Be specific and evidence-based
- Avoid subjective or emotional language
- Focus on the code, not the person
- Provide actionable feedback
